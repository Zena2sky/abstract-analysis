## Excess Vocabulary Lists
본 폴더에는 KCI 인문·사회과학 초록(2004–2024) 분석을 통해 도출된
과잉 어휘(excess vocabulary) 목록이 언어 및 분야별로 정리되어 있습니다.

### 파일 구성

각 파일은 해당 언어(English/Korean) 및 분야(Social Sciences/Humanities)에 대한
2024년 기준 과잉 어휘 목록을 포함합니다.

### 과잉 어휘 선정 기준

과잉 어휘는 다음 절차에 따라 선정되었습니다.

1. 2021–2022년 문서 빈도 추세를 기반으로 2024년 예측 빈도(q)를 선형 외삽(linear extrapolation)으로 산출
2. 2024년 실제 빈도(p)와 예측 빈도(q)를 비교하여 다음 두 지표 계산

   - **Delta (p − q)** : 절대적 증가 폭
   - **Ratio (p / q)** : 상대적 증가율

3. 2023년과 2024년 모두에서 전체 문서의 0.01% 이상 등장한 단어만 분석 대상에 포함
4. Delta 상위 100개, Ratio 상위 100개를 각각 추출
5. 두 집합의 교집합을 제거한 후 결합하여 최종 과잉 어휘 목록 구성

### 분석 목적

Delta는 기존에 일정 수준 사용되던 표현의 **절대적 확대**를 포착하며,
Ratio는 기존 사용 빈도가 낮았던 표현의 **상대적 급증**을 포착합니다.

두 지표를 병행함으로써 서로 다른 유형의 문체 변화를 구분하여 분석할 수 있습니다.
