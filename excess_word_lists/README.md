## Excess Vocabulary Lists

본 폴더에는 영어 사회과학, 영어 인문학, 
한국어 사회과학, 한국어 인문학 초록 분석 결과가 각각 별도의 파일로 제공됩니다.

각 파일은 2024년을 기준으로 delta(p−q) 상위 100개 단어와 ratio(p/q) 상위 100개 단어를 각각 추출한 후, 
두 집합의 중복을 제거하여 결합한 목록입니다.

각 행에는 연도별 관측 비율(p2021–p2024), 
2024년 예측 비율(q2024), 
과잉 지표(delta, ratio), 로그 변환값(log10_ratio), 
지표별 순위(delta_rank, ratio_rank) 등이 포함됩니다.
