## Code

본 폴더에는 KCI 인문·사회과학 초록(2004–2024) 분석에 사용된
전처리, 빈도 계산, 과잉 어휘 선별 및 시각화 코드가 포함되어 있습니다.

### 현재 구성

#### 1. 단어 수준 분석(논문 3,4장) 
- 영어 초록 분석 코드 (시각화 포함)

  ※ 사회과학과 인문학은 동일한 분석 절차를 사용하며, 입력 데이터 변경을 통해 재현 가능합니다.

- 한국어 초록 분석 코드

  ※ 영어 분석과 동일하나 형태소 분석 및 패턴 복원 단계 추가

#### 2. 초록 단위 확산 분석(논문 5장) 
- 평균·중앙값 및 분포 이동 분석

  ※ 해당 코드는 영어/한국어 및 사회과학/인문학 데이터에 공통으로 적용됩니다.

## 재현성 관련 사항

- 예측값(q)은 2023년을 제외한 2021–2022년 추세를 기반으로 산출되었습니다.
- 모든 시각화 결과는 `/figures` 폴더에 정리되어 있습니다.
- 최종 과잉 어휘 목록은 `/excess_word_lists` 폴더에 정리되어 있습니다.
